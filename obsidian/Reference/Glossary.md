# Glossary

| Term | Meaning |
|---|---|
| **Atlas** | The Sliderobes internal operations platform (this project) |
| **Anthill** | External CRM — source of sales, leads, payments → [[Integrations]] |
| **Xero** | External accounting system — invoices & payments → [[Financial Flow]] |
| **WorkGuru** | External project/scheduling system; source of customer IDs |
| **PNX** | Proprietary board-cutting machine file format |
| **OS Door** | Outsourced door component for an order |
| **PO** | Purchase Order |
| **AP** | Accounts Payable (email-to-invoice pipeline via MS Graph) |
| **GL Code** | General Ledger code — chart-of-accounts category in Xero |
| **Fit / Fitter** | Installation / the installer who fits the furniture |
| **Fit Board** | Visual calendar for scheduling fits |
| **Contract number** | Primary key linking orders across Anthill, Xero, and Atlas |
| **Par level** | Stock threshold that triggers a stock take |
| **Remedial** | Warranty / corrective work order |
| **Coversheet** | Digital job coversheet (`SaleCoverSheet`) |
| **Raumplus** | A material/ordering brand with its own validation rules |
