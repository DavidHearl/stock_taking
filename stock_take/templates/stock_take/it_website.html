Sliderobes Website

Get the WordPress site running locally — install the tools, clone the repo, and preview your changes.

Before you start. These commands are for Windows PowerShell. Run each block in order. If a winget command isn't recognised, download the app from the linked site instead — the result is the same.
  1. 1

    Install Visual Studio Code

    The code editor you'll use to browse and edit the site. Install it with the command below, or download it from code.visualstudio.com. 

    winget install --id Microsoft.VisualStudioCode -e
  2. 2

    Install Git

    Git is the tool that downloads (clones) the website's code and lets you pull future updates. Install it, then close and reopen PowerShell so the git command becomes available. 

    winget install --id Git.Git -e

    Set your name and email once so your changes are attributed correctly: 

    git config --global user.name "Your Name"
git config --global user.email "you@sliderobes.com"
  3. 3

    Clone the repository

    This downloads the site into a sliderobes folder inside your Documents. You'll be prompted to sign in to GitHub the first time — use your GitHub account (or a personal access token if the repo is private). 

    cd $HOME\Documents
git clone https://github.com/DavidHearl/sliderobes.git

    To pull the latest changes later, run git pull from inside the sliderobes folder.

  4. 4

    Set up a local WordPress environment

    The site is WordPress/PHP, so it needs PHP, MySQL and a web server to run. The easiest option is Local (by WP Engine) — it bundles everything and creates a local site in a few clicks. Install it with: 

    winget install --id WPEngine.Local -e

    In Local, create a new site, then point its app/public (or wp-content) folder at the files you cloned in step 3 — or copy the cloned theme/plugin folders into the new site's wp-content. Prefer XAMPP instead? Install PHP + MySQL with XAMPP and serve the folder from htdocs.

  5. 5

    Open the project in VS Code

    Open the cloned folder in VS Code to start editing. When prompted, install the recommended PHP extension for syntax highlighting and hints. 

    code $HOME\Documents\sliderobes

    Recommended extensions — install from the Extensions panel (Ctrl+Shift+X) or run: 

    code --install-extension bmewburn.vscode-intelephense-client
code --install-extension wordpresstoolbox.wordpress-toolbox
Stuck on any step? Raise a ticket in Atlas and include the exact command you ran and the error message.
